### 🌸 IEEE Conference Paper

**“Intelligent Road Safety System: Integrating Machine Learning for Enhanced Speed Regulation and License Plate Verification”** 

✨ **Overview:**  
Developed an intelligent road safety system that detects speeding violations using average speed tracking and license plate recognition.

📍 Published in IEEE Conference Proceedings  

✨ **Key Focus:**
